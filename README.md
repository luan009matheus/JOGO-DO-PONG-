# 🕹️ Pong com Loja

Um remake do jogo Pong com loja e personalização.

## ✅ Recursos

- Loja com skins para raquetes e bola
- Sistema de pontuação
- Personalização visual
- IA inimiga simples

## 🎮 Controles

- `W` / `S` - Mover a raquete do jogador
- `L` - Abrir loja
- `ESC` - Sair da loja
